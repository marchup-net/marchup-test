Install locust using pip install locust

Change the constants in constant.py

    Set testSpaceId to the id of the space you are using to test
    set username and password to your username and password

start locust by running the "locust" command in the directory with locustfile.py
go to the web interface

change number of users to total number of users you want to test at max load
change ramp up to number of users added per second
change host to the api url and start
